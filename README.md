# 推し活共有地図 Oshi Map
## サイト概要
### サイトテーマ
自分の推しアニメ、マンガ、役者、芸能人、YouTuberの聖地を検索したり、共有できるSNS
​
### テーマを選んだ理由
自身が旅行が好きで推し活とは違う目的で旅行に行くことが多いのですが、旅行先に知っている作品の撮影地やモデルになっている建物などのことを後から知ることが多くありました。  
また現在推し活という言葉が世間的にも広く認知されるようになっていますが、その中の聖地の情報については特定の作品のみに特化した情報がSNSやブログに分散され体系的に管理されていません。  
こうした中で地図に直接聖地の情報を保存してお互いに共有できるサービスがあれば便利だと考えました。

### ターゲットユーザ
-コンテンツに対して熱心な推しを持つファン  
-イベント情報や聖地の足跡を地図上に記録し推しへの熱量を可視化したい人  

### 主な利用シーン
-自分の推しコンテンツの聖地を探すとき  
-自分が行った聖地をシェアするとき  
-旅行に行くときにチラッと聖地を確認しておきたいとき
